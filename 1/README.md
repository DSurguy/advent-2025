# Advent of Code 2025 - Day 1
https://adventofcode.com/2025/day/1

# Setup
Download https://adventofcode.com/2025/day/1/input and place it into this folder named `input` (no extension)

# Running
- From the project root, run `bun run 1`
- From this directory, run `bun index.ts`

# Checking output
The puzzle solutions are output directly to stdout, while the detailed log of spins and adjustments is written to a human-readable CSV format in `out.csv`. Feel free to grab your favorite spreadsheet tool and explore!
